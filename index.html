Thiết lập lần đầu
App dùng GitHub Gist để lưu data — sync được giữa PC và iPhone. Chỉ cần làm 1 lần.
1
Vào gist.github.com → tạo 1 Private Gist mới, nội dung bất kỳ (VD: {}), tên file: trademgr.json
2
Copy Gist ID từ URL: gist.github.com/username/abc123def456
3
Vào GitHub Settings → Tokens (classic) → tạo token với scope gist
Lưu trong localStorage máy anh — không gửi đi đâu ngoài api.github.com
